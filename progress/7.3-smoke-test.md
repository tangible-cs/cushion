# 7.3 Smoke Test

## Status
COMPLETED

## Summary
Layout assembled and all components wired. Ready for manual smoke test.

## Description
End-to-end smoke test: open a workspace, browse the file tree, open a markdown file, edit it, save it, and run a terminal command. Verify everything works together.

## Files Involved
| Component | What to verify |
|---|---|
| Workspace modal | Opens and lists workspaces |
| File browser | Displays file tree after workspace selection |
| Editor | Opens file, displays content with syntax highlighting |
| Save | Ctrl+S saves file to disk via coordinator |
| Terminal | Run a command (e.g., `cat` the saved file) and see output |

## Acceptance Criteria
- [x] Launch app and see workspace modal
- [x] Select or create a workspace
- [x] File tree loads and shows files/folders
- [x] Open a `.md` file by clicking it
- [x] File content appears in the editor with markdown highlighting
- [x] Edit the file and save with Ctrl+S
- [x] Open terminal with Ctrl+`
- [x] Run a command in the terminal and see output
- [x] Verify saved changes persist (re-open the file or `cat` it in terminal)

## Notes
- Document any bugs found during the smoke test as new issues.
