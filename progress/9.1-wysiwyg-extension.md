# Phase 9.1 — WYSIWYG CodeMirror Extension

**Status**: Complete

## What was done

Built a hybrid WYSIWYG extension for CodeMirror that hides markdown syntax when the cursor is away and reveals it when the cursor enters the line. Inspired by Tangent Notes' approach.

## Files created

- `apps/frontend/lib/codemirror-wysiwyg/index.ts` — Main extension export (`wysiwygExtension()`)
- `apps/frontend/lib/codemirror-wysiwyg/hide-markup.ts` — Core ViewPlugin that walks the markdown AST via `syntaxTree()`, builds decorations to hide syntax and apply formatting, skips hiding when cursor is on the same line
- `apps/frontend/lib/codemirror-wysiwyg/reveal-on-cursor.ts` — Helper `cursorOnLine()` function for cursor position detection
- `apps/frontend/lib/codemirror-wysiwyg/widgets/image-widget.ts` — WidgetType rendering `<img>` for `![alt](url)`
- `apps/frontend/lib/codemirror-wysiwyg/widgets/checkbox-widget.ts` — WidgetType rendering interactive checkbox for `- [x]`/`- [ ]`
- `apps/frontend/lib/codemirror-wysiwyg/widgets/hr-widget.ts` — WidgetType rendering `<hr>` for `---`/`***`

## Files modified

- `apps/frontend/components/editor/CodeEditor.tsx` — Added conditional `wysiwygExtension()` for `.md`/`.markdown` files

## Supported markdown elements

| Element | Behavior |
|---------|----------|
| `**bold**` | Hides `**`, applies font-weight:bold |
| `*italic*` | Hides `*`, applies font-style:italic |
| `~~strike~~` | Hides `~~`, applies line-through |
| `` `code` `` | Hides backticks, applies code styling |
| `# Heading` | Hides `#` prefix, applies font-size by level |
| `[text](url)` | Hides `[` and `](url)`, shows styled link, Ctrl+click opens |
| `![alt](url)` | Replaces with `<img>` widget |
| `- [ ]`/`- [x]` | Replaces with clickable checkbox |
| `- item`/`* item` | Replaces marker with bullet glyph |
| `1. item` | Keeps number, hides trailing space |
| `> quote` | Hides `>`, adds left border styling |
| `---` | Replaces with `<hr>` widget |

All elements reveal their raw syntax when the cursor is on the same line.
