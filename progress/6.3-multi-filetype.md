# 6.3 Multi-Filetype Support

## Status
COMPLETED

## Summary
CodeEditor.tsx detects file extension and applies the correct CodeMirror language mode: markdown, javascript/jsx, typescript/tsx, json, css, html, python. Falls back to plain text for unknown extensions.

## Description
Support opening any text file, not just markdown. Detect file type from extension and apply the appropriate syntax highlighting mode. Markdown is the primary mode, but also support JSON, JavaScript, TypeScript, CSS, HTML, YAML, and others.

## Files Involved
| File | Role |
|---|---|
| `apps/frontend/src/components/editor/CodeEditor.tsx` | Language detection and mode switching built-in |
| `apps/frontend/package.json` | Language mode packages added |

## Acceptance Criteria
- [x] File extension is detected when opening a file
- [x] Correct syntax highlighting applied per file type
- [x] Supported types: `.md`, `.json`, `.js`, `.ts`, `.jsx`, `.tsx`, `.css`, `.html`, `.py`
- [x] Unknown file types open with plain text mode (no highlighting)
- [x] Switching between files updates the language mode

## Notes
- Component renamed from MarkdownEditor to CodeEditor.
